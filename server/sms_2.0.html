Example settings ...
From = your cell # --> e.g. 4165550000
Who0 = contact nickname --> e.g. Bob
Num0 = email-to-txt addr --> e.g. 4165551111@sms.rogers.com

Check the email-to-txt addr with your hotmail/gmail ... account!!
If you receive txts by sending an email to the email-to-txt addr from hotmail/gmail ..., the same email addr should work with this app.
The key is to determine this part "@sms.rogers.com" for the person you're trying to reach.

Note: You can always use your hotmail/gmail account as the email-to-txt addr.
Doing so will show you have the settings correctly configured and any issue is with the "@sms.carrier.com" part.
